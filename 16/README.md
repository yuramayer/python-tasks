# Скользящее среднее по уникалам

Я обрабатываю поток событий вида

```python
(user_id, ts, value)
```

В любой момент времени нужно ответить:

> Каково среднее значение `value` среди
уникальных пользователей,
которые совершили хотя бы одно событие
за последние 10 минут?

Решение **инкрементальное**: при поступлении
нового события не должно пересчитываться
всё окно
