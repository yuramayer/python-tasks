# Комнаты и встречи

Дан список интервалов (start, end) - начала и окончания встреч (или задач). Каждая встреча должна идти непрерывно от start до end в одной комнате (на одном воркере).

Нужно посчитать **минимальное количество комнат**, чтобы провести все встречи без перекрытий в одной комнате.

## Пример

Входные данные

```python
[(0, 30), (5, 10), (15, 20), (30, 40), (35, 45)]
```

Ожидаемый ответ: 2
