# Складываем

Даны два непустых связных списка, каждый из которых представляет неотрицательное число,
где цифры записаны в обратном порядке (т.е. первый узел — младший разряд).

Нужно сложить эти два числа и вернуть результат также в виде связного списка в том же формате (обратный порядок)

Время = O(max(m, n)) - один проход по списка
Память = О(max(m, n)) = для нового списка
