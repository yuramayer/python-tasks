# Воркеры

Дан список заданий, которые необходимо выполнить (task_name, dur, delay_max)

У каждого задания есть длительность, и оно должно запуститься на позднее delay_max.

Определить для заданного числа воркеров N, можно ли выполнить задания не позднее указанного времени.

Пример входных данных:

```text
('task1', 100, 20)
('task2', 100, 40)
('task3', 100, 50)
('task4', 100, 100)
```

Окей, то есть мы передаём `N` и `tpl` - tuple.

У каждой задачи есть:

- `dur` - время, сколько задача занимает условных единиц (секунд, минут)
- `delay_max` - задача должна стартовать не позже чем через `delay_max` условных единиц от начала планировать.
- `N` - количество воркеров, которые работают параллельно
