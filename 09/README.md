# Merge Sorted Array

Тебе даны два целочисленных массива nums1 и nums2,
отсортированных по неубыванию, и два числа m и n,
обозначающие, сколько элементов реально содержится в каждом из них.

Нужно объединить оба массива в один отсортированный прямо внутри
nums1, не создавая новый.

У nums1 длина m + n: первые m элементов - это данные,
которые нужно объединить,
а последние n - просто нули-заглушки, чтобы хватило места.
nums2 имеет длину n.

Требуется добиться результата за O(m + n).
