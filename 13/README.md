# Слияние интервалов

Дан список интервалов `[(start, end)]`, `start < end`

Нужно объединить все пересекающиеся интервалы в один, сохранить порядок и не потерять не пересекающиеся

## Пример 1

```python
data = [(1,3), (2,6), (8,10), (15,18)]
```

Результат:

```python
[(1, 6), (8, 10), (15, 18)]
```

## Пример 2

```python
data = [(1, 4), (4, 5)]
```

Результат:

```python
[(1, 5)]
```
